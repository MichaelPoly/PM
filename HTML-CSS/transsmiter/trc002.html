Автомобильный крэдл для телефона - идеальный вариант для большинства телефонов. Он имеет два варианта крепления - к решетке воздуховода, либо на лобовое стекло. Развдижной механизм работает в двух направлениях - по горизонтали и по вертикали.

Еще одним важным преимуществом можно отметить самозажимной механизм: как только вы прислоняете телефон к устройству, зажимы по горизонтали мнгновенно фиксируют его.

Комплектация:

1. Зарядное устройство

2. Кабель USB-microUSB

3. Крепление к лобовому стеклу

4. Крепление к воздуховодам отопителя

5. Адаптер питания 12В - 5В в прикуриватель

6. Вспомогательный диск для крепления ЗУ на торпеду автомобиля.
